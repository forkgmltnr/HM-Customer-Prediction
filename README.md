# H&M-Customer-Prediction
* 프로젝트 주제: H&M 고객 이탈 예측 분석
* 프로젝트 목적: 고객 이탈 예측 하여 충성 고객 증대 위한 방법 제시 - 고객 이탈 예측을 통해 기업의 고객 유출 예측 하고 고객 생애 가치(CLV) 증대 마케팅 전략 제시
* 프로젝트 기간: 2023.09 - 2023.10
